# Connor.ai

Connor.ai is a multi-agent frontier intelligence daily report system built around AgentScope 2.0.

It is designed to collect, evaluate, cluster, track, write, review, and archive AI / semiconductor / tech-finance intelligence with full evidence lineage and trace replay.
